Анализ Voyant tools

1) Выводит количество слов, которые употребляются в тексте (частота)
2) Лексическая плотность
3) Readability index (https://www.analyzemywriting.com/readability_indices.html#:~:text=A%20readability%20index%20is%20an,well%20readers%20comprehend%20the%20text) Нужно будет использовать несколько методов и взять среднюю из них
4) Среднее количество слов в предложении
5) Выводит топ самых встречаюзихся слов в корпусе
6) Есть функция загрузки текста или же URL
7) Выводит коллокации 
8) Выводит контекст
9) Строит графики тенденций упоминаний в котором по У (Относительне частотные повторения) а по Х сегменты документа
10) Выводит связь слов через график TermsBerry
11) Строит карту ссылок
12) Запоминает фразы
13) Понимает ЮРЛ и берет из него весь текст
14) Строит график распространения для каждого из слов
